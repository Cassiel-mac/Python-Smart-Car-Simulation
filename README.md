# Python-Smart-Car-Simulation

本科智能小车 Python 仿真学习项目  
从0基础系统学习 Python + 控制理论 + 仿真能力

## 项目目标
- 掌握 Python 在科学计算和控制仿真中的使用
- 复现并升级本科智能小车（循迹 + 避障）
- 逐步学习现代控制理论（状态空间、PID、观测器、故障诊断）
- 为后续四旋翼无人机仿真项目打下坚实基础

## 项目路线图（6个阶段）

### **阶段 1：Python 基础与科学计算（Week 1-2）**
- Python 环境配置（Anaconda + Jupyter）
- 基础语法、函数、类
- NumPy（数组、矩阵运算）
- Matplotlib（可视化绘图）

### **阶段 2：小车运动学建模（Week 3-4）**
- 二轮差速小车运动学模型
- 坐标系转换与仿真
- 键盘/程序控制小车运动
- 数据记录与可视化

### **阶段 3：控制算法实现（Week 5-6）**
- PID 控制器设计与调参
- 循迹控制实现
- 避障逻辑（传感器模拟）
- 实时动画可视化

### **阶段 4：进阶控制与观测（Week 7-8）**
- 状态空间表示法
- 卡尔曼滤波器入门
- 传感器噪声处理
- LQR 控制器初步尝试

### **阶段 5：故障诊断与容错（Week 9-10）**
- 执行器故障注入
- 残差生成与故障检测
- 简单容错控制
- 与第一篇 IEEE 论文方法对接

### **阶段 6：项目收尾与扩展**
- 完整项目报告
- 代码重构与模块化
- 过渡到四旋翼模型（后续项目）
